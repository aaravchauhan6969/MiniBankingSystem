# 💰 Mini Banking System

## 📌 Description

Mini Banking System is a full-stack project that allows users to perform basic banking operations such as creating an account, depositing money, withdrawing money, checking balance, and viewing transaction history.

This project integrates:

* Frontend (HTML, CSS, JavaScript)
* Backend (Node.js)
* Core Logic (C using Data Structures)

---

## 🎯 Features

* Create Account
* Deposit Money
* Withdraw Money
* Check Balance
* Transaction History
* Undo Last Transaction

---

## 🧠 DSA Used

* Linked List → Store accounts
* Stack → Undo transactions
* Queue → Transaction history
* Searching → Find accounts

---

## 🛠 Technologies Used

* HTML, CSS, JavaScript
* Node.js, Express.js
* C Programming (DSA)
* File Handling (accounts.txt)

---

## 🚀 How to Run

### 1. Compile C Code

```
gcc bank.c -o bank
```

### 2. Start Server

```
node server.js
```

### 3. Open Frontend

Open `login.html` in browser
