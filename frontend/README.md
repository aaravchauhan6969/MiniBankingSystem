# 🎨 Frontend - Mini Banking System

## 📌 Description

This is the user interface of the Mini Banking System built using HTML, CSS, and JavaScript.

---

## 💡 Features

* User-friendly interface
* Login system
* Banking operations (Deposit, Withdraw, Balance, etc.)
* Displays output from backend

---

## 🛠 Technologies Used

* HTML → Structure
* CSS → Styling (Premium UI)
* JavaScript → API calls & interaction

---

## 🚀 How to Run

* Open `login.html` in any browser
* Login using credentials
* Use the banking dashboard

---

## 📌 Note

Make sure backend server is running before using the frontend.
